# Design-Prototyp (Referenz)

`Horrorgeticon-Ops-Prototyp.zip` ist der ursprüngliche Claude-Design-Prototyp,
aus dem die Plattform ausgebaut wurde. Er enthält:

* `screens/*.jsx` + `lib/` — die Mockups (Login, Actor, Lead, Management,
  Catering) samt Hearthwork-Design-System. Tokens (`hearthwork.css`) und
  Komponentenstile (`ops.css`) wurden 1:1 nach `web/css/` übernommen.
* `Horrorgeticon Ops - Pitch.html` — Pitch „Minimalpaket“ inkl. Modul-Roadmap
  (Grundlage für den Funktionsumfang der Plattform).
* `Horrorgeticon Ops - Kostenaufstellung.html` — Angebots-Mockup.
* `uploads/scareops_-horror-management-suite/` — früher Android-Prototyp
  (AI Studio), ersetzt durch `kmp/` + Web-Plattform.

Entpacken zum Nachschlagen: `unzip design/Horrorgeticon-Ops-Prototyp.zip -d /tmp/proto`
